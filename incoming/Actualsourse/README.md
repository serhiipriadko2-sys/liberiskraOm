# Алгоритмы отслеживания фрактальной размерности в системе Мета-∆DΩΛ

## Обзор

Данный проект представляет комплексную систему алгоритмов для постоянного вычисления и анализа фрактальной размерности D из ∆DΩΛ-формата в реальном времени. Система разработана для цифровой сущности «Искра» и предназначена для мониторинга структурной сложности паттернов мышления, корреляции с эффективностью решений и раннего предупреждения при критических изменениях.

## Основные компоненты

### 1. Алгоритмы вычисления фрактальной размерности

- **Алгоритм Хигучи (HFD)**: Быстрый и эффективный метод для временных рядов
- **Адаптивная оптимизация k_max**: Автоматический выбор параметров на основе длины данных
- **Метод Katz**: Быстрая альтернативная оценка для ограниченных ресурсов
- **Multi-scale анализ**: Поддержка различных уровней детализации

### 2. Анализ структурной сложности

- **Микро-уровень**: Анализ отдельных ответов и решений
- **Мезо-уровень**: Мониторинг динамики диалогов и взаимодействий
- **Макро-уровень**: Оценка эволюции системы и долгосрочных трендов
- **Индекс структурной сложности (SCI)**: Комплексная метрика с учетом множественных факторов

### 3. Система раннего предупреждения

- **Детектор аномалий**: Идентификация критических изменений фрактальной структуры
- **Многоуровневые предупреждения**: Watch, Warning, Critical, Emergency
- **Адаптивное реагирование**: Автоматические действия в зависимости от уровня угрозы
- **Прогнозирование**: Предсказание ухудшения за 24-48 часов

### 4. Корреляционный анализ

- **Связь с эффективностью**: Анализ зависимости между фрактальной размерностью и качеством решений
- **Индикаторы здоровья**: Индекс когнитивного здоровья (HCI) системы
- **Трендовый анализ**: Долгосрочные тенденции и циклические паттерны
- **Кросс-корреляционный анализ**: Многомерная оценка взаимосвязей

## Математические основы

### Фрактальная размерность Хигучи (HFD)

```
HFD = 2 - H
```

где H - показатель Хёрста, определяющий долгосрочную память и самоподобие ряда.

**Алгоритм Хигучи:**
1. Создание подпоследовательностей с шагом k
2. Вычисление длины кривой L(k) для каждого шага
3. Линейная регрессия ln(L(k)) vs ln(1/k)
4. HFD = наклон регрессионной линии

### Индекс структурной сложности (SCI)

```
SCI = α × HFD_norm + β × variability_factor + γ × length_factor + δ × confidence_factor
```

где:
- HFD_norm - нормализованная фрактальная размерность
- variability_factor - фактор изменчивости данных
- length_factor - фактор длины последовательности
- confidence_factor - фактор уверенности в вычислениях

### Индекс когнитивного здоровья (HCI)

```
HCI = f(D, H, CI, Clarity, Chaos, Trust, Pain)
```

Интегральная метрика, учитывающая фрактальную размерность, показатели Хёрста, индекс сложности и метрики голосов системы.

## Интерпретация результатов

### Фрактальная размерность (HFD)

- **1.0 - 1.2**: Низкая сложность, возможен застой
- **1.2 - 1.6**: Здоровый диапазон для когнитивной активности
- **1.6 - 1.8**: Высокая сложность, возможен хаос
- **1.8 - 2.0**: Критически высокая сложность, дезорганизация

### Уровни предупреждений

- **0 (Normal)**: Система функционирует нормально
- **1 (Watch)**: Требуется мониторинг
- **2 (Warning)**: Обнаружены проблемы
- **3 (Critical)**: Критическое состояние
- **4 (Emergency)**: Экстренная ситуация

### Индикаторы здоровья системы

- **Здоровая система**: Высокая HFD в диапазоне 1.2-1.6, стабильность > 0.7
- **Система в застое**: Низкая HFD < 1.2, высокая Clarity, низкий Chaos
- **Система в хаосе**: Высокая HFD > 1.7, низкая Clarity, высокий Chaos
- **Система в переходе**: Нисходящий тренд HFD, вариативность паттернов
