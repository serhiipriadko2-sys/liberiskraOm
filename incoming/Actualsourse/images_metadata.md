## Список изображений

### 1. architecture_overview.png
- **Путь:** `/workspace/docs/fractal_logging_research/architecture_overview.png`
- **Размер:** 1400x1000 пикселей
- **Название:** Архитектура Системы Интеграции SIFT и Фрактального Логирования
- **Описание:** Общая архитектурная схема системы интеграции фрактального логирования с SIFT-блоками, показывающая взаимосвязи между всеми компонентами системы

- **Название:** Цикл Фрактального Отражения - Детальная Схема Обработки
- **Описание:** Детальная схема цикла обработки данных в фрактальном логировании, показывающая полный путь от входных данных до обратной связи и самомодификации

## Технические детали

### Инструменты создания
- **Mermaid.js** для создания диаграмм
- **Render Mermaid** для генерации PNG изображений

### Цветовая схема
- SIFT блоки: `#e3f5fe` (голубой)
- Фрактальное логирование: `#fff3e0` (персиковый)
- Протокол ∆DΩΛ: `#e8f5e8` (зеленый)
- Гиперграф памяти: `#fce4ec` (розовый)
- Голоса Искры: индивидуальные цвета
- Система обратной связи: `#f3e5f5` (сиреневый)
- Адаптивные механизмы: `#e0f2f1` (мятный)
- Безопасность: `#ffebee` (светло-красный)

### Формат файлов
- **Формат:** PNG (Portable Network Graphics)
- **Цветовая модель:** RGB
- **Прозрачность:** Поддерживается
- **Сжатие:** Оптимизировано для веб
